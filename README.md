# E-Commerce Recommendation System

## Overview
A Streamlit-based recommendation system using **Python, Django, Flask, and MongoDB**.

## Installation
1. Clone the repository
2. Install dependencies using `pip install -r requirements/requirements.txt`
3. Run the application using `streamlit run main.py`

## Features
- **Top Products**: Displays popular products.
- **Textual Clustering**: Groups similar products based on descriptions.
- **Collaborative Filtering**: Suggests products based on purchase history.
